# ProyectoEmiliano
Este proyecto consiste en el desarrollo de una página web dedicada a la Diablada Pillareña, una de las festividades culturales más importantes del Ecuador, celebrada en el cantón Píllaro.

## Ejecución rápida
Abre `/home/runner/work/ProyectoEmiliano/ProyectoEmiliano/index.html` en un navegador.

## Funcionalidades incluidas
- Secciones informativas sobre historia, tradiciones y personajes de la Diablada Pillareña.
- Contenido interactivo para explorar personajes representativos.
- Registro de usuarios con almacenamiento local.
- Reproducción de música tradicional mediante reproductor web.
- Visualización de vestimentas y máscaras en formato de galería.
- Apartado administrativo local para gestionar publicaciones informativas.
